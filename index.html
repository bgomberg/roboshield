Roboshield is a robotics oriented shield for the Arduino Mega.

{{ '### Features' | markdownify }} {{ '* 14 analog inputs' | markdownify }} {{ '* up to 7 digital I/O pins' | markdownify }} {{ '* 4 x 2Amp DC motor outputs \(DRV8848 H-bridge\)' | markdownify }} {{ ' * 2 of the outputs support Pololu 6-pin motors with software support for reading encoders' | markdownify }} {{ '* 8 RC servo outputs' | markdownify }} {{ ' * servo power regulated to 5V' | markdownify }} {{ '* 1 software controlled LED' | markdownify }} {{ '* 16x2 LCD' | markdownify }} {{ '* battery voltage monitoring' | markdownify }} {{ '* 7-15V input voltage' | markdownify }} {{ '* on/off switch' | markdownify }} {{ '* protection' | markdownify }} {{ ' * reverse power input protection' | markdownify }} {{ ' * input pins protected against high voltage application' | markdownify }} {{ '* support for the GY-521 accelerometer/gyroscope breakout board' | markdownify }} {{ ' * readily available on Ebay' | markdownify }} {{ '* external servo and motor supply capable' | markdownify }}
